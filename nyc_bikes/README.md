# BigQuery SQL Analysis – NYC Citi Bike Stations

## Overview
This project showcases SQL queries written and executed in **Google BigQuery** using the public dataset  
`bigquery-public-data.new_york.citibike_stations`.  
The analysis focuses on understanding Citi Bike station distribution, capacity, and geographic patterns across New York City.

## Dataset
- **Source:** Google BigQuery Public Datasets  
- **Table:** `bigquery-public-data.new_york.citibike_stations`  
- **Description:** Contains metadata about Citi Bike stations, including location, capacity, and operational status.

## Key Analysis Areas
- Station distribution by city and neighborhood  
- Bike capacity and dock availability analysis  
- Identification of high-capacity and low-capacity stations  
- Geographic insights using latitude and longitude data  

## Skills Demonstrated
- SQL (Standard SQL – BigQuery)  
- Data aggregation and filtering  
- Use of CTEs for query structuring  
- Exploratory data analysis on real-world public datasets  
- Writing readable, well-documented analytical queries
